# CGOA - Certified GitOps Associate

[![CGOA](https://img.shields.io/badge/CNCF-CGOA-326CE5?style=for-the-badge&logo=git&logoColor=white)](https://training.linuxfoundation.org/certification/certified-gitops-associate-cgoa/)

The **Certified GitOps Associate (CGOA)** exam demonstrates knowledge of GitOps principles, practices, and related technologies.

## Exam Overview

| Detail | Information |
|--------|-------------|
| **Exam Format** | Multiple Choice |
| **Number of Questions** | 60 |
| **Duration** | 90 minutes |
| **Passing Score** | 75% |
| **Certification Validity** | 3 years |
| **Cost** | $250 USD |
| **Retake Policy** | 1 free retake |

## Exam Domains & Weights

| Domain | Weight |
|--------|--------|
| GitOps Terminology | 20% |
| GitOps Principles | 30% |
| Related Practices | 16% |
| GitOps Patterns | 20% |
| Tooling | 14% |

## Key Topics

### GitOps Principles
- Declarative configuration
- Version controlled
- Automated delivery
- Software agents

### GitOps Patterns
- Repository structures
- Environment promotion
- Secret management
- Multi-tenancy

### Tooling
- Flux CD
- Argo CD
- Git workflows
