# CGOA Demo Exam

Welcome to the **Certified GitOps Associate (CGOA)** demo exam! This interactive practice exam will help you prepare for the actual certification.

## About This Exam

- **Total Questions**: 68 questions covering all CGOA domains
- **Format**: Multiple choice questions
- **Passing Score**: Understanding of GitOps principles and practices
- **Time**: Self-paced practice

## Exam Domains

The questions are organized across these key domains:

1. **GitOps Terminology** - Core concepts and definitions
2. **GitOps Principles** - The four fundamental principles
3. **GitOps Patterns** - Common implementation patterns
4. **GitOps Tooling** - Flux, Argo CD, and related tools
5. **GitOps Workflow** - Day-to-day operations
6. **GitOps and Kubernetes** - Integration with K8s
7. **Security in GitOps** - Security best practices
8. **CI/CD and GitOps** - Integration patterns
9. **Infrastructure as Code** - IaC with GitOps
10. **Deployment Strategies** - Blue-green, canary, etc.
11. **Observability and Monitoring** - Tracking and debugging
12. **Troubleshooting GitOps** - Common issues and solutions
13. **Compliance and Audit** - Governance and compliance
14. **Collaboration and Workflow** - Team practices
15. **Getting Started with GitOps** - Initial setup
16. **GitOps Overview** - High-level concepts
17. **Related Practices** - DevOps, IaC, DevSecOps

## Tips for Success

1. **Understand the Four Principles** - Declarative, Versioned & Immutable, Pulled Automatically, Continuously Reconciled
2. **Know Pull vs Push** - Understand why pull-based is preferred
3. **Practice with Tools** - Hands-on experience with Flux or Argo CD
4. **Study Security** - Secret management and security best practices
5. **Learn Patterns** - Repository structures and deployment strategies
